# AXIOM v2.0.0 Public Release Draft

## LinkedIn / Blog Version

AXIOM v2.0.0 is live.

This release turns AXIOM from an experimental prototype into a measurable local reasoning core:

- Stable public API contract across `learn`, `ask`, `verify`, `reason`, `compare`, and `dream`
- `paranoidMode` for zero-external-LLM operation
- `AXIOM_ERROR` and `contractVersion` for programmatic integrations
- MCP stdio adapter for Claude Desktop, Cursor, and other tool-aware clients
- Deterministic benchmark fixtures so performance can be tracked commit to commit

Why it matters:
AXIOM is built for deterministic, evidence-based symbolic reasoning without forcing everything through a cloud model.

Repo: https://github.com/agiulucom42-del/axiom
